# Code of Conduct

This project is open\-source, community driven, and intentionally informal. The goal is to be welcoming and constructive while keeping collaboration safe and productive.

Scope
- Applies to contributions and interactions in `issues`, `pull requests`, `discussions`, repository comments, chat related to this project, and project events.

Our values
- Be respectful and assume good intent.
- Focus feedback on code, not individuals.
- Use inclusive language and avoid slurs, hateful or harassing speech.
- Help others when you can; welcome newcomers.

Unacceptable behavior
- Harassment, threats, hate speech, slurs, or demeaning language.
- Doxxing, sharing private or sensitive information.
- Targeted or repeated abusive behavior.
- Sexual harassment or unwanted sexual content.
- Deliberate sabotage, spamming, or malicious tampering.

Non\-affiliation notice
- This project uses some assets and a name that were previously associated with another organization. The `README.md` clarifies that this project is now independent and not affiliated with that organization.

Reporting
- To report a problem, open a new issue titled `Code of Conduct report` and add the label `conduct`. Include relevant context and any evidence (links, screenshots, timestamps). If you prefer less visibility, mention it in the report and maintainers will handle it discreetly.
- You can also mention or notify the maintainers directly on GitHub (for example by mentioning `@TavstalDev`).

Response and enforcement
- Maintainers will acknowledge reports within 72 hours.
- Possible actions: request for change, temporary restrictions on contributions, removal of content, or permanent ban. Severe cases (threats, doxxing) may be blocked immediately and, if necessary, referred to law enforcement.
- Appeals: create a new issue titled `Code of Conduct appeal` and include relevant context; maintainers will review.

Tone and intent
- This is not meant to be corporate or legalistic. The goal is simple: keep the project open, collaborative, and safe for everyone.
- If you're unsure whether something fits, ask or err on the side of respectful behaviour.

Thank you for contributing and helping keep this project welcoming.
